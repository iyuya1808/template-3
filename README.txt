===================

このファイルの用途: デプロイ方法・編集ポイントの説明書

## 概要
「麺匠 雅」は、やや高級感のあるラーメン店のウェブサイトです。
モダンなデザインとレスポンシブ対応で、お客様に上質な体験を提供します。

## ファイル構成
- index.html     : トップページ（店舗紹介・キャッチ・予約導線）
- menu.html      : メニューページ（ラーメン・サイド・ドリンク）
- gallery.html   : ギャラリーページ（店内・料理写真）
- contact.html   : アクセス・予約ページ（地図・フォーム・FAQ）
- styles.css     : 全ページ共通スタイルシート
- scripts.js     : JavaScript（ナビ・フォーム・ギャラリー機能）
- README.txt     : このファイル

## デプロイ方法

### 1. 静的ホスティングサービス
以下のサービスで簡単にデプロイできます：

**Netlify（推奨）**
1. https://netlify.com にアクセス
2. ファイル一式をZIPで圧縮
3. ドラッグ&ドロップでアップロード
4. 自動的にURLが発行されます

**Vercel**
1. https://vercel.com にアクセス
2. GitHubリポジトリまたは直接アップロード
3. 自動デプロイ設定

**GitHub Pages**
1. GitHubリポジトリを作成
2. ファイルをアップロード
3. Settings > Pages で公開設定

### 2. 独自ドメイン設定
- ドメイン取得後、DNSでCNAMEレコードを設定
- SSL証明書は各サービスで自動取得

### 3. 必要な追加設定
- Google Maps API（地図表示用）
- フォーム送信先の設定（Netlify Forms等）
- Google Analytics（アクセス解析）

## カスタマイズ方法

### 店舗情報の変更
**基本情報**
- 店名: 全ファイルの「麺匠 雅」を検索・置換
- 住所: contact.html と各ページフッター
- 電話番号: contact.html と各ページフッター
- 営業時間: contact.html と各ページフッター

**メタ情報（SEO）**
- 各HTMLファイルの <title> タグ
- <meta name="description"> の内容
- Open Graph情報（og:title, og:description等）

### デザインの変更
**カラーパレット（styles.css の :root セクション）**
```css
:root {
    --bg-primary: #111117;      /* メイン背景色（黒） */
    --bg-secondary: #f8f6f0;    /* サブ背景色（クリーム） */
    --accent-primary: #bfa57a;   /* アクセント色（ゴールド） */
    --text-primary: #333333;     /* メインテキスト色 */
}
```

**フォント変更**
- Google Fonts のリンクを変更
- CSS変数 --font-primary, --font-heading を更新

### メニューの編集
**menu.html で以下を編集：**
- メニュー名と価格
- 説明文
- 画像URL（プレースホルダーを実際の写真に）
- カテゴリの追加・削除

### 画像の差し替え
**現在はプレースホルダー画像を使用**
1. 実際の写真を撮影
2. 適切なサイズにリサイズ（推奨：800x600px程度）
3. WebP形式で圧縮（パフォーマンス向上）
4. HTMLの src 属性を更新
5. alt テキストも適切に設定

**推奨画像サイズ**
- ヒーロー画像: 1200x800px
- メニュー写真: 400x300px
- ギャラリー: 800x600px
- 店主写真: 400x400px

### フォーム送信先の設定
**現在はフェイク処理**
実際の運用では以下を設定：

1. **Netlify Forms（簡単）**
   - form タグに netlify 属性を追加
   - 自動でフォーム処理

2. **外部サービス**
   - Formspree, EmailJS等
   - scripts.js の submitReservation 関数を修正

3. **独自API**
   - バックエンドサーバーを構築
   - AJAX通信でデータ送信

### 地図の埋め込み
**contact.html の地図プレースホルダーを置換**
```html
<!-- 現在 -->
<div class="access__map-placeholder">...</div>

<!-- Google Maps埋め込み例 -->
<iframe src="https://www.google.com/maps/embed?pb=..." 
        width="100%" height="300" frameborder="0"></iframe>
```
